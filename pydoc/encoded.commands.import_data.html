encoded.commands.import_data module¶

The inpath is the path to either:

  • a zip file or directory containing single sheet xlsx/tsv/csv files named after the object type, e.g. experiment.xlsx, antibody_lot.tsv
  • a single Excel workbook with worksheets named after the object types.

Examples.

To load the initial data:

%(prog)s –attach ../dccMetadataImport/data/documents/
–attach ../dccMetadataImport/data/ENCODE3docs/ ../documents-export.zip dev-masterdata.ini

To make updates from a single tsv file

%(prog)s –username ACCESS_KEY_ID –password SECRET_ACCESS_KEY
–patch ../updates/ http://localhost:6543
