encoded.audit.biosample module¶

encoded.audit.biosample.audit_biosample_culture_date(value, system)[source]¶

A culture_harvest_date should not precede a culture_start_date. This should move to the schema.

encoded.audit.biosample.audit_biosample_depleted_term_match(value, system)[source]¶

The depleted_in_term_name and depleted_in_term_name should be concordant. This should be a calcualted field. If one exists, the other should. This should be handled in the schema.

encoded.audit.biosample.audit_biosample_donor(value, system)[source]¶

A biosample should have a donor. The organism of donor and biosample should match. Pooled_from biosamples do not need donors??

encoded.audit.biosample.audit_biosample_gtex_children(value, system)[source]¶

GTEX children biosamples have to be properly registered. - aliases (column A from plate-maps) - part_of pointing to the parent biosample - source Kristin Ardlie

encoded.audit.biosample.audit_biosample_human_no_model_organism_properties(value, system)[source]¶

human bioamples shouldn’t have model organism properties initiated

encoded.audit.biosample.audit_biosample_part_of_consistency(value, system)[source]¶
encoded.audit.biosample.audit_biosample_subcellular_term_match(value, system)[source]¶

The subcellular_fraction_term_name and subcellular_fraction_term_id should be concordant. This should be a calculated field If one exists the other should. This should be handled in the schema.

encoded.audit.biosample.audit_biosample_term(value, system)[source]¶

Biosample_term_id and biosample_term_name and biosample_type should all be present. This should be handled by schemas. Biosample_term_id should be in the ontology. Biosample_term_name should match biosample_term_id.

encoded.audit.biosample.audit_biosample_tier1_cell_lines_have_required_cell_culture_properties(value, system)[source]¶

tier1 cell lines have required fields from biosample_cell_culture that must be present and that depend on the cell_line

encoded.audit.biosample.audit_biosample_transfection_type(value, system)[source]¶

A biosample with constructs or rnais should have a transfection_type
