Get data for nested properties - Award and Lab. Save to state or use state object in callback.

setLinkedDetails([fallbackToAjax][, callback][, newState])¶
Arguments:
  • fallbackToAjax (boolean) – Whether to revert to AJAX to fetch info if not in provided ExpSet object.
  • callback (function) – A function ran after execution, before any AJAX fetching, which takes ones parameter - an object representing resulting state change. Used in place of setState if have more state to apply.
  • newState (Object) – State object to use. Defaults to empty object.

Inner getEmbeddedPropertyDetailsFromExperiments¶

Using similar approach(es) as browse.js to grab property details from experiments_in_set. Grabs property detail object from sub-experiment(s) rather than embedding in own JSON output to keep size of ExperimentSets down as can’t remove (for example) experiments_in_set.lab from encoded/types/experiment.py > ExperimentSet as it’d affect Browse page experiment filtering & view.

getEmbeddedPropertyDetailsFromExperiments(propertyName, allowAjaxFallback)¶
Arguments:
  • propertyName (string) – Name of property/key in schema for which to get details.
  • allowAjaxFallback (boolean) – Whether to start an AJAX request to fetch details if they are not in experiment(s).
Returns Object:

Details for the property/key supplied, or null if not available or not matched.
