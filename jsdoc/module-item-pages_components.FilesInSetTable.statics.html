Description¶

Static functions which help to render or parse File objects.

DPROPERTIES

PROPS

Functions¶

atIdFromObject¶

Convert a link_id, if one exists on param ‘object’, to an '@id‘ link.

atIdFromObject(object)¶
Arguments:
  • object (Object) – Must have a ‘link_id’ or '@id‘ property. Else will return null.
Return string|null:
 

The Item’s '@id‘.

attachmentDownloadLinkFromFile¶

Generate a download link for a file attachment from a fileObject, which should represent a (partial) JSON of a file Item.

attachmentDownloadLinkFromFile(fileObject)¶
Arguments:
  • fileObject (Object) – Object must have a link_id or an @id property, as well as an ‘attachment’ property containing ‘href’.
Return string:

URL to download file attachment from.

iconClassFromFileType¶

Converts a file-type or mime-type string into a FontAwesome icon className suffix.

iconClassFromFileType(fileType)¶
Arguments:
  • fileType (string) – MIMEType to get icon className suffix for.
Return string|null:
 

The suffix to append to “fa-” or “icon-” CSS class.
