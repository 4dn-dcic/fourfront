Component: module.exports.Aggregator¶

Member Of Module: viz/BarPlot

This is an optional component which may be placed between BarPlot.Chart and UIControlsWrapper. It will store the result of aggregation into state and then pass it as a prop down to BarPlot.Chart. Primarily this is to redrawing performance. Utilizes shouldComponentUpdate and componentWillReceiveProps.

Accepts the same props as BarPlot.Chart, save for own ‘aggregatedData’ and ‘aggregatedFilteredData’.
