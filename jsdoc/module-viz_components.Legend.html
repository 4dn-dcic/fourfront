Props¶

fields¶

Type: Array.<Object>

List of objects containing at least ‘field’, in object dot notation. Ideally should also have ‘name’.

includeFieldTitle¶

Type: boolean

Whether to show field title at top of terms.

className¶

Type: string

Optional className to add to Legend’s outermost div container.

width¶

Type: number

How wide should the legend container element (<div>) be.

title¶

Type: string | Element | Component

Optional title to display at top of fields.
