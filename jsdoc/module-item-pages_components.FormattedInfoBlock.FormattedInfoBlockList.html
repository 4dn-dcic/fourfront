Component: FormattedInfoBlockList¶

Optional container of FormattedInfoBlocks, wrapping them in a <UL> and <LI> elements. Encapsulates all required ajax/aggregation for fetching array of fields/data. Available via FormattedInfoBlock.List

Props¶

details¶

Type: Array.<Object>

Array of complete details to display.

endpoints¶

Type: Array.<string>

Array of endpoints to AJAX details from.

renderItem¶

Type: function

Render function for items. Should return a FormattedInfoBlock component.

fallbackMsg¶

Type: Component | Element | string

What to display if both details and endpoints don’t exist or are empty.

propertyName¶

Type: string

Descriptive unique ID of property/ies displayed.

ajaxCallback¶

Type: function

Callback to execute with details, if/after they are fetched w/ AJAX.
