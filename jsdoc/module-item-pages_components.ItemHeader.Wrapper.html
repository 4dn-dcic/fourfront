Component: Wrapper¶

Use this to wrap ItemHeader.TopRow, .MiddleRow, and .BottomRow to create a complete ItemHeader. Passes own props.context and props.href down to children.

Local Navigation

  • Props

Props¶

context¶

Type: Object

Same as the props.context passed to parent ItemHeader component.

href¶

Type: string

Location from Redux store or '@id‘ of current Item. Used for View JSON button.

schemas¶

Type: Object

Pass from app.state. Used for tooltips and such.
