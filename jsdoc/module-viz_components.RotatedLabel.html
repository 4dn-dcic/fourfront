Component: RotatedLabel¶

Member Of Module: viz/components

A label meant to be place along an X-axis. Given an angle, label text, placementHeight, and other properties, calculates visible portion of label and rotates it. Handles showing full label onHover. Optionally adds a directional pointer icon to tail of label.

Props¶

angle¶

Type: number

Angle of label rotation. Defaults to 30.

placementWidth¶

Type: number

How wide along the X axis is the object to be labeled. Defaults to 60.

placementHeight¶

Type: number

How much height, +~10px, is available for the label(s). Defaults to 50.

isMounted¶

Type: boolean

Pass true if we are not server-side and/or if parent component is mounted. Will not render anything unless this is true. Defaults to false.

lineHeight¶

Type: number

Line height of label, in px. Defaults to 14. Recommended to leave 14px as CSS stylesheet is set up for this value.

deRotateAppend¶

Type: boolean

If true, caret or other appendage will be de-rotated from props.angle. Defaults to 14.

append¶

Type: Element | Component | string

A React Element or Component to append at tail of label, centered at bottom of X axis placement. Defaults to an upwards-pointing caret.
