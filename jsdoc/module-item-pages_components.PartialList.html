Component: PartialList¶

Member Of Module: item-pages/components

Renders a list using elements along the Bootstrap grid. Takes two lists as props: ‘persistent’ and ‘collapsible’. Persistent items are always visible, while collapsible are only shown if props.open is true.

Props¶

persistent¶

Type: Array.<Component> | Array.<Element> | Array.<string>

React elements or components to always render.

collapsible¶

Type: Array.<Component> | Array.<Element> | Array.<string>

React elements or components to render conditionally.

open¶

Type: boolean

Show collapsed items or not.

className¶

Type: string

Class name for outermost element.

containerType¶

Type: string

Type of element to use as container for the two lists. Defaults to ‘div’.
